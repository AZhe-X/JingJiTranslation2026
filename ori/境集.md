# 境集

这个页面整理收集了一些作者的文章，大体上勾勒了作者的核心哲学框架，提供给有兴趣的读者。
虽然这些文字写于数个不同的时间段，并且不按照时间排列。但是建议读者按照这个页面给定的顺序阅读。

## 上集：鱼

这部分文章以随笔的形式，记录下了作者对于一些自己困惑的思考。虽然建议按照顺序阅读，但是每一篇文章也可以拿出来单独阅读。

1. [[关于集体利益的一点点思考 其一]]
2. [[A Thought on Knowledge]]
3. [[西西弗斯、班与猫]]


## 下集：渔

这部分文章概括了上集文章背后，作者所使用和发展的方法论。作者用自己的语言，较为系统性的记录了自己的哲学。文章所用的概念和名词同哲学传统有一定差异，需要注意甄别以免混淆。不同于上集，这几篇文章关联性较强，建议将它们一同完整阅读之后联系理解。由于这几篇文章是正在工作中的草稿，有许多未完成的部分，读者可以直接跳过他们。

1. [[后形而上学 其一]]
2. [[形而上学后的目的：诗意]]
3. [[范畴论与境]]
4. [[范畴论与境：K1策略]]
